Step 7. Add an unordered list

We will now use an unordered list to create a navigation bar. An unordered list (as opposed to an ordered list, which would be numbered) consists of an opening <ul>, and a series of <li></li> tags. The list items (hence “li”) will go inside of the <li> tags.

For your website, add an unordered list after your “Dehydrated Water, Inc.” header. Between the <ul> tags, add three <li> tags, and put in “Home,” “Products,” and “Contact Us” inside. Now use Live Preview. Your website should match Figure 2.1. If not, check out the code in Figure 2.2. Step 8. Create a stylesheet This is great, but it is not very aesthetically appeasing. To change the default ul styling (or any other HTML element) we need to use Cascading Style Sheets. Now this is where we create an external stylesheet. Create a new file called “main.css” in the same folder as your “index.html.” At the top of the document, under the title tags but still inside of the head section, add the following: <link rel= “stylesheet” type= “text/css” href= “main.css”/> . The rel= “stylesheet” and type= “text/css” means that the link is referring to a stylesheet, and href= “main.css” tells the browser to look for the stylesheet in a file called “main.css.” The <link/> tag is one of those self-closing tags that I mentioned earlier. Step 9. Learn CSS in 113 seconds The syntax of CSS is completely different from HTML: there are no tags, and we do not specify the language type at the top of the document. For CSS, we simply identify the HTML element which we are styling (“the selector”), and between curly braces, define the values for our intended properties. For example, if we do h1{ color: blue;} all h1 content will have a color of blue. Notice the semi-colon after the value blue; if this is not present, it will cause an error. To select an HTML element, we can either target the tags which is enclosed in (which is undesirable in most cases, as all other content enclosed in those tags will have that styling), or we can assign it a class in the HTML document and declare the properties in an external style sheet (or we can also give it an “id,” which is essentially the same thing as a class, but it can only be used once per document). Step 10. Start using CSS Navigate to main.css, and type the following: ul{ display: inline; list-style: none; } The “ul” means we are selecting all of the unordered lists on the document (which is fine for now), “list-style: none;” will cause the bullet points to disappear, and “display: inline” will make the list horizontal.
