How to Make Your First Website From Scratch

<part 1>

Websites are made of HTML, CSS, and JavaScript files which browsers convert into understandable information. Hyper Text Markup Language (HTML) provides the content and structure of a webpage (all of the text and images and their order), Cascading Style Sheets (CSS) adds style (font colors, backgrounds, and how everything fits together) and JavaScript adds behavior (the ability to fade in text, allow the user to change content, and even make games). By the end of this tutorial, you will have made a simple website for free. And unlike those who use sites such as Wix for web development, you will understand how websites work.

Step 1. Get Brackets, a text editor

To make a website, you need to code. No way around it. (Unless, of course, you use Wix, but that's lame.) Fortunately, the learning curve for web development is sharp, and there are copious amounts of help online (including, of course, this tutorial). My personal go-to’s are Stack Overflow, a Q&A forum of millions of professional programmers, and w3 schools, an awesome web development resource. I had less than a month of web development experience when I made this website.

I personally use WebStorm by JetBrains, but without a free student subscription this would cost over a hundred dollars per year. So for your website, we will use Brackets, a free text editor. To get started, go to http://brackets.io and download the program.

Step 2. Create a HTML file named “index.html”

When you first open Brackets, you will be presented with an example project, which just so happens to be their own homepage. Ignore this for now.

Instead, look to the grey panel on the left. Under “Getting Started,” right-click and select “New Folder.” Name this folder “myfirstwebsite.” Right click on “myfirstwebsite” and select “New File.” Name this “index.html.” Congrats, you now have an empty HTML file!

When a browser goes to a website, it will first look for a file called “index.html.” This is the “home page” of a website, and it is the standard practice to name it this way.

Step 3. Edit “index.html”

Add <!DOCTYPE html> to the top of the page. This needs to be on the HTML page, and simply tells the browser that the following HTML file will be in the HTML 5 format, the most recent version of HTML.

Step 4. Learn some fundamentals

For a browser to properly display a website, there a number of “tags” (special words enclosed in angle brackets such as <h1> and <body>) which need to be present. All of the website’s content will be enclosed in tags, and the specific tag will determine the effect. For example, the <strong> tag in <strong>Bold Text</strong> will cause “Bold Text” to be in bold.

Notice the pattern: an opening tag, the content, and the closing tag, albeit with a /. All opening tags must have a matching closing tag, or else the website will not look as intended. (However, there are some tags which are self-closing, which we will return to later.)

This is the central concept of HTML, and once you understand this, mastering HTML will be learning the roughly twenty tags.

Step 5. Add <html>, <head>, and <title> tags

The next item you will add will be <html>. This tag will enclose the entire HTML document, and tells the browser that the information in between the tags is HTML code.

Did you see what happened? After you typed <html>, Brackets automatically created the closing tag. This is a nice feature, and will help prevent errors.

Now, in between the two <code><html></code> tags, add the <code><head></code> tag. The content in the <code><head></code> tag will not actually be displayed on the web page, but is important nonetheless, as it contains other data, such as the name of the website.

Inside of the <code><head></code> tags, add <code><title></code>, and inside of the title tags, add “Dehydrated Water Inc.” Make sure that the title tag is indented. This will make your code more readable.

Refer to Figure 1.1. This is what you should have so far.

Now look at the lightning bolt in the top right corner. When clicked, this will open the current HTML page in your default browser. This is called “Live Preview,” and will allow us to make changes and see the result. Click on the lightning bolt and see what happens.

You should see nothing, except that in the tab it says, “Dehydrated Water, Inc.” This is the purpose of the title tag, to display the website’s name.

Step 6. Add to the document body

Now, go below the head tag (but still inside of the <code><html></code> tags), and add the <code><body></code> tag. Everything inside of the body tag will be visible on the web page.

For the first tag inside of the document body, add the <code><div></code> tag. This creates a new section within the document, and we will add content in here which we want to be together. (The div tag does a few other things, such as allow us style all of the elements within it, which I will discuss later.) Press enter a few times to create empty lines between the div tags.

Between the div tags, add a <code><h1></code>, and inside of the <code><h1></code>, type “Dehydrated Water, Inc.” H1 stands for “header 1,” and makes the text bold and larger than usual (the default is 12pt). There are five other headers, running up to H6, and each decrease in font size.

Now click on the lightning bolt. You should see “Dehydrated Water, Inc” at the top of the page. If you don’t, refer to Part 1 Recap for what your code should look like.
